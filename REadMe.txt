Резюме: В этом задании мы собираемся немного повеселиться с нашим встроенным веб-приложением ресторана. Первая страница нашего веб-приложения содержит 3 кликабельных плитки: Меню, Специальные предложения и Карту. Если вы нажмете на плитку Специальные предложения, то попадете на одну страницу категории, где будут показаны все пункты меню, относящиеся к категории Специальные предложения. Ваша задача в этом задании состоит в том, чтобы изменить это поведение таким образом, чтобы, когда пользователь нажимает на плитку Специальных предложений, веб-приложение выводит пользователя на случайную страницу меню одной категории, перечисляя пункты меню в категории, будь то "Обед", "Ужин", "Суши" и т. Д. Таким образом, любая случайная категория может стать Специальными предложениями! Как весело! (нет! :-) )

Чтобы сделать это, нам нужно изменить домашний HTML-фрагмент и, помимо динамического извлечения его с сервера, также вставить случайную категорию " short_name` в вызов функции следующего кода. Ранее код для отправки пользователя в категорию "Специальные предложения" был таким:

`html
<a href="#" onclick="$dc.loadMenuItems('SP');">
```

Для этого задания мы изменили эту строку, чтобы подготовить ее к случайной категории "short_name", которая будет такой:

`html
<a href="#" onclick="$dc.loadMenuItems({{randomCategoryShortName}});">
```

**ПРИМЕЧАНИЕ: предоставленный код не будет запущен. Это зависит от вас, чтобы следовать инструкциям, чтобы заставить его работать.**

6. Вы не имеете права изменять `home-snippet.html` файл. Любые изменения значения свойства ' randomCategoryShortName` должны быть сделаны в коде Javascript.

7. Есть 4-5 довольно простых шагов для реализации необходимого функционала.
* Откройся `js/script.js` файл.
* Найдите TODO: ШАГ 0 и следуйте инструкциям, пока не закончите с TODO: ШАГ 4.
<br><br>
Если вы смотрели лекции, код должен быть вам хорошо знаком.<br><br>

8. Как только вы закончите, убедитесь, что желаемая функциональность работает правильно. Используйте синхронизацию браузера или разверните свое решение на страницах GitHub.
* Загрузите домашнюю страницу в браузер.
* Нажмите на плитку Специальных предложений. Должна появиться одностраничная категория со списком пунктов меню для **какой-то** категории.
* Нажмите на логотип ресторана, чтобы вернуться на главную страницу.
* Снова нажмите на плитку Специальных предложений. Скорее всего, будет показана другая страница категории одной страницы.

Повторите это несколько раз, чтобы убедиться, что большую часть времени вы видите другую страницу одной категории.
